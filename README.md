# taller-tp-grupal
Trabajo practico final Taller Fiuba

# Dependencias
    coreutils
    cmake
    build-essential
    qt5-default
    libsdl2-dev
    libsdl2-image-dev
    libsdl2-ttf-dev
    liblua5.3-dev
    libyaml-cpp-dev
    libsdl2-mixer-dev

Instalar con el siguiente comando en una terminal:
sh dependencies.sh

# Para compilar
    mkdir build
    cd build 
    cmake ..
    make

# Cada nueva clase agregada
    cd build
    cmake ..
    make
